# 🏦 Loan Prediction System

A Machine Learning web application that predicts whether a loan application is likely to be **Approved** or **Rejected** using a **Random Forest Classifier**.

## 🚀 Live Demo

🔗 **https://loan-prediction-system-6bpddt86fkbo39qx6mndpy.streamlit.app/**

## 📂 GitHub Repository

🔗 https://github.com/Syeda-Madani/Loan-Prediction-System

## ✨ Features

- Predicts Loan Approval or Rejection
- User-friendly Streamlit interface
- Real-time prediction
- Confidence score displayed
- Random Forest Machine Learning Model

## 🛠️ Technologies Used

- Python
- Pandas
- NumPy
- Scikit-learn
- Streamlit
- Random Forest Classifier
- Git & GitHub

## 📁 Project Files

- `app.py` – Streamlit application
- `Loan_Prediction.ipynb` – Model training notebook
- `loan_model.pkl` – Trained Machine Learning model
- `requirements.txt` – Project dependencies
- Encoder `.pkl` files

## ▶️ How to Run

1. Clone the repository.
2. Install the required packages:

```bash
pip install -r requirements.txt
```

3. Run the application:

```bash
streamlit run app.py
```
